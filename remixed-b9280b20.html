Provisioners para agentes de IA

Smallstep recomienda certificados de corta duración y mTLS en lugar de API keys. Seleccioná un provisioner para ver el flujo y cuándo usarlo.

JWK

JSON Web Key

Más versátil

Token JWT generado programáticamente desde el código del agente. Ideal para integraciones custom sin dependencia de infraestructura específica.

Cloud / K8sSA

AWS · GCP · Azure · K8s

Ideal en cloud

Valida la identidad del agente usando documentos de identidad nativos de la infraestructura. Sin secretos adicionales que gestionar.

IAMInstance identityK8s SA

ACME Device Attestation

Hardware-bound

Máxima seguridad

Vincula criptográficamente el acceso del agente a hardware confiable (TPM, Secure Enclave). El certificado no puede migrar a otro dispositivo.

TPMAttestHardware
